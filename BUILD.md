# Build Instructions

## Overview

This repository contains only the release builds and installation files for the password manager application. The source code is maintained in a private repository.

## Pre-built Releases

Pre-compiled binaries are available in the [Releases](../../releases) section. Download the appropriate installer for your operating system.

## Building from Source

The source code for this project is not publicly available. If you are an authorized contributor with access to the private source repository, please refer to the internal documentation for build instructions.

## System Requirements

- **Operating System:** Windows 10/11 (64-bit)
- **RAM:** 4 GB minimum (8 GB recommended)
- **Storage:** 100 MB free disk space
- **.NET Runtime:** .NET 6.0 or later (included in installer)

## Questions

For questions regarding the build process or contributing, please open an issue or contact the maintainers.
